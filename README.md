# TiendaWeb - Proyecto de Gestión de Eventos

## 1. Levantar la Base de Datos
Necesitas tener Docker Desktop instalado. En la terminal, dentro de la carpeta `tiendaweb-backend`:
```bash
docker-compose up -d

2. Levantar el Backend
En una terminal nueva, dentro de tiendaweb-backend:

dotnet run

3. Levantar el Frontend
En otra terminal, dentro de tiendaweb-frontend:

Bash
npm install
ng serve


4. Acceso
API: http://localhost:5056

Web: http://localhost:4200
